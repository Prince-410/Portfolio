# Personal Portfolio

A modern, responsive, and interactive personal portfolio website showcasing professional experience, advanced technical skills, and completed projects. Designed with a clean aesthetic and optimal user experience in mind.

## 🚀 Features

- **Modern UI/UX:** Clean, elegant glassmorphism and modern aesthetic.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Elements:** Smooth scrolling, functional contact forms, and engaging micro-animations.
- **Project Showcase:** Highlights key projects with links and technical details.
- **Performance Optimized:** Fast loading times and minimal dependencies.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Custom styling, animations, and responsive layouts (Vanilla CSS).
- **JavaScript (ES6+):** Interactive logic, DOM manipulation, and dynamic data fetching.
- **Vercel:** Deployment and hosting configuration.

## 📂 File Structure

- `index.html`: Main HTML document detailing the layout.
- `style.css`: All styling, including typography, colors, and animations.
- `script.js`: Interactive functionality and asynchronous operations.
- `vercel.json`: Deployment configuration for Vercel.

## 🔧 Getting Started (Local Development)

To run this project locally, simply clone the repository and open the `index.html` file in your preferred browser, or use a local development server like Live Server (VS Code extension).

```bash
git clone https://github.com/Prince-410/Portfolio.git
cd Portfolio
```
